# GenSpark AI란?

### All-in-One AI Workspace

**GenSpark**은 AI 기반 검색에서 출발하여, 현재는 **AI 워크스페이스**로 진화한 플랫폼입니다.

| 기존 방식 | GenSpark 방식 |
|-----------|--------------|
| 검색 따로, 문서 따로, 개발 따로 | 하나의 플랫폼에서 모두 해결 |
| 도구마다 계정·학습 필요 | 통합 인터페이스, 자연어 명령 |
| 결과물 연결이 어려움 | 검색 → 문서 → 앱 연결 가능 |

**설립:** 2023년 (전 Baidu 부사장 Eric Jing 창업)
**특징:** Mixture-of-Agents 아키텍처

---

# GenSpark 핵심 도구 한눈에 보기

| 도구 | 기능 | 활용 예시 |
|------|------|----------|
| **AI Chat** | AI 대화 & 검색 | 리서치, 질문 답변 |
| **AI Developer** | 웹사이트/앱 생성 | 랜딩페이지, 웹앱 |
| **AI Slides** | 프레젠테이션 생성 | 강의 자료, 발표 |
| **AI Sheets** | 스프레드시트 자동화 | 데이터 분석, 보고서 |
| **AI Docs** | 문서 작성 | 기획서, 보고서 |
| **AI Designer** | 디자인 에이전트 | 로고, 배너, UI |
| **AI Image** | 이미지 생성 | 일러스트, 사진 |
| **Super Agent** | 자동화 에이전트 | 복합 작업 자동화 |

# AI Developer - 작동 원리

### Mixture-of-Agents (MoA) 아키텍처

```
사용자 프롬프트
    ↓
┌──────────────────────┐
│   중앙 오케스트레이터   │  ← 작업을 분석하고 분배
└──────────┬───────────┘
           ↓
  ┌────────┼────────┐
  ↓        ↓        ↓
┌────┐  ┌────┐  ┌────┐
│디자인│  │로직 │  │코드 │   ← 전문화된 AI 모델들
│모델 │  │모델 │  │모델 │
└────┘  └────┘  └────┘
  ↓        ↓        ↓
  └────────┼────────┘
           ↓
   완성된 웹사이트/앱
```

**핵심:** 하나의 AI가 아닌, **여러 전문 AI가 협업**하여 최적의 결과 도출

# AI Developer - 3단계 프로세스

### Step 1: Describe (설명하기)
- 자연어로 아이디어 한 줄 설명
- 예: *"AI 교육 서비스 랜딩페이지를 만들어줘"*

### Step 2: Generate (생성하기)
- AI가 자동으로 설계, 코딩, 디자인 수행
- 실시간으로 진행 과정 확인 가능

### Step 3: Publish (배포하기)
- 원클릭으로 라이브 URL 생성
- GenSpark 서브도메인 즉시 배포
- (선택) 커스텀 도메인 연결 가능

<div class="highlight-box">

**소요시간:** 간단한 랜딩페이지 = 약 **5~15분**

</div>

# 데모 포인트 1: 프롬프트의 힘

### 프롬프트 품질이 결과물 품질을 결정합니다

| 나쁜 프롬프트 | 좋은 프롬프트 |
|--------------|-------------|
| "홈페이지 만들어줘" | "AI 교육 서비스 랜딩페이지를 만들어줘" |
| 결과: 기본 템플릿 | + 포함할 섹션을 구체적으로 나열 |
| 수정 많이 필요 | + 디자인 스타일 지정 |
| | + 색상/분위기 명시 |

### 좋은 프롬프트의 3요소:
1. **무엇을** 만들 것인지 (목적)
2. **어떤 내용**이 들어갈 것인지 (구성요소)
3. **어떤 느낌**으로 (디자인/톤)
